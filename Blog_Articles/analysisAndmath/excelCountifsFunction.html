Overview

Microsoft Excel COUNTIFS function examines and returns the total number of cells which meet a specific requirement. COUNTIFS could be based on some criteria viz; dates, text. It also executes logical operators such as ><, = and wild

Goal

To find the total number of cells that meet a certain measure

The value returned

The frequency of meeting criteria

Formula

=COUNTIFS (range1, criteria1, [range2], [criteria2] ...)

The Logic

•range1 - The first range to be assessed.

•criteria1 - The criterion which is to be used on range1.

• range2 - [elective] the second range to be assessed.

• criteria2 - [elective] the criterion which is to be used on range2.

When to use

COUNTIFS determine the array of cells for which certain conditions are met. COUNTIFS could execute multiple conditions with more than a range. The use of ranges and criteria is in a pair. For an extra condition, some other range and criteria must be provided. The total range and criteria can be up to 127 range/criteria

Illustration

The illustration below indicates that COUNTIFS can be used to evaluate series or entries with two criteria.

=COUNTIFS(C5:C15,"blue",D5:D16,"PH") // blue and PH
=COUNTIFS(C5:C15,"Pink",F5:F15,">10") // Pink and >10

Hints:

• All supplementary ranges should possess the exact count of rows and columns as the first range. The array does not necessarily need to be beside. Mismatch arrays will result in #VALUE errors.

• When criteria are not in number format, then a double quotation mark should be inserted while criteria in number format do not require the quotation mark. For instance: 200, “200”, ”40”, or B2. B2 contains number

• Wildcards including (?*) can feature in criteria while question marks fit any character. An asterisk can pair a series of characters. Use an asterisk (*), or a tilde just before a question mark to obtain plain question mark or asterisk

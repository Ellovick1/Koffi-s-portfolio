The formula

=LOOKUP (2,1/(Item="Pen"),price)

Explanation

To look up the current price listing for a commodity, with recent items entered at the rear. In such situations, the LOOKUP function can be used. For instance; the formula in G7 is as scheduled below

=LOOKUP(2,1/(item=F7),price)

Here, the array B5: B11 represents the commodities while D5: D11 is the price

To use the formula

The Excel LOOKUP is a built-in function which returns an entry from an array; either a column or row. The function pre-supposes an accurate ordering of entries for pairing. If the LOOKUP value is the biggest value in the LOOKUP range, the custom operation is to return to the previous value. This formula executes this process by generating a range containing 1s and error only. It purposely searches for the number 2 which will never be established.

LOOKUP scouts the range for 2, since it is not established; it returns to the previous value which is 1 to replace the content at similar spot in the outcome range.

Initially, this statement is expressed as

Item=F7

where f7 contains “Pencil“, the outcome is

{FALSE; TRUE; FALSE;TRUE;FALSE;FALSE;TRUE;FALSE}

This number is then used to divide 1 to supply the ultimate LOOKUP range

{#DIV/0!;1;#DIV/0!;1;#DIV/0!;#DIV/0!;1;#DIV/0!}

The LOOKUP function searches for 2, then returns the final 1(the seventh spot in the range) and replaces the seventh commodity on the LOOKUP range with price $30. To read more about the concept of intentionally looking for a value that won't ever appear, read about.
